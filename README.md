AlphCord,
 Custom Discord theme for Vencord / BetterDiscord
 Clean • Minimal • Custom branding

 Features,
Custom name display (top-left),
Clean & modern UI tweaks,
Compatible with Vencord & BetterDiscord,
Lightweight (CSS only),

 Installation,
Install Vencord or BetterDiscord,
Go to Themes,
Click Open Themes Folder,
Drop AlphCord.theme.css inside,
Enable the theme,
